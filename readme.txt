1) Для подключения добавить на страницу:
<script src="https://vistegra.by/support/support.js"></script>

Этого достаточно, что бы чат работал с общим ботом.

2) Настройки (не обязательно). Для настроект используется глобальная переменная:
<script>
    window.supportBotSettings = {...}
</script>

Доступные свойства:
bot - строка - ключ токена бота, узнать доступные значения по ftp в папке support/
btn - строка - селектор или тег кнопки.
position - строка - Расположение кнопки и окна. Доступные значения: 'topLeft', 'topRight', 'bottomLeft', 'bottomRight'.
from - строка | функция - Строка или результат фукнции, которое будут видеть в чате от кого пришло сообщение.
